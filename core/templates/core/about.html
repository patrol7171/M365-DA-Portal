About This Site

The DevPatrol Data Analytics Portal is a prototype of an internal company site that allows multiple users to access and utilize one web application for their data analytics information and visualizations. DevPatrol is a fictitious company and this site is built off of an Office365 developer sandbox/test environment.

The "view" for each user will differ since it is based on the user's team association. In order to test this site, please contact the developer for a list of test user credentials.
